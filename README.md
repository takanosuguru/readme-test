# Gun Mod - Q&A

## English

**Q: How do I fire my weapon?**  
A: Hold down the *Action 2* key for the basic pistol or sniper rifle distributed on the Discord Gladio Mori server until the weapon fires.

**Q: I pressed the key but it didn't fire.**  
A: If you press the key immediately after the match starts or right after firing once, the bullet won't fire. In that case, release the key for one to two seconds, then press it again.

**Q: Where is the bullet generated and fired from?**  
A: It is generated based on the front of your hips. For example, if your hips are facing downward, the bullet will be generated facing downward.

**Q: Bullets don't fire in the test range or locally.**  
A: This MOD only works in a multiplayer environment.

## 日本語

**Q, どうしたら武器が発射されますか？**  
A, Discord「Gladio Mori」サーバーで配布している basic pistol または sniper のアクション2キーを、発射されるまで押し続けてください。

**Q, 押したのに発射されません。**  
A, 試合開始直後や一度発射した直後にキーを押しても弾は出ません。その場合は1〜2秒ほどキーから指を離してから、再度押してください。

**Q, どこを基準に弾が生成され、発射されるのですか？**  
A, ヒップ（腰）の正面を基準に生成されます。たとえばヒップが下を向いている場合、弾も下向きに生成されます。

**Q, テスト場やローカルでは弾が出ません。**  
A, このMODはマルチプレイ環境でのみ動作します。
